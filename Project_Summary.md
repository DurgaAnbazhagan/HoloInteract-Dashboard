
     HoloInteract Dashboard – Project Summary
Project Title:-
HoloInteract Dashboard – IoT Device Management System
Objective:-
The objective of this project is to develop a responsive frontend dashboard for managing HoloInteract IoT devices installed across multiple customer locations. The dashboard enables business owners to monitor device status, view campaign analytics, and visualize important performance metrics through a clean and user-friendly interface.
Features Implemented:
Login Page:-
Responsive login interface
Email and password fields
Clean UI design
Device Dashboard:-
Device Name
Device Location
Online/Offline Status
Last Active Time
Device Search functionality
Analytics Dashboard:-
Total Devices
Active Devices
Total Interactions
Campaign Success Percentage
Weekly Device Activity Chart

Bonus Features:
AI Device Failure Prediction:-
Predicts possible device failures based on historical device performance, helping reduce unexpected downtime.
Smart Content Recommendation & Engagement Heatmap:-
Analyzes customer interaction across locations and recommends the most effective content for each device.
Remote Device Health Monitoring:-
Displays battery level, network connectivity, storage usage, and overall device health from a centralized dashboard.
Technologies Used:-
HTML5
CSS3
JavaScript
Font Awesome
Conclusion:-
This project demonstrates the design and implementation of a modern IoT dashboard prototype using frontend technologies. It focuses on usability, responsive design, and scalable feature ideas while meeting all internship assignment requirements.
